
# Exploring Gemini

### create virtual environment 
`virtualenv .venv`

### activate virtualenv 
`source ./.venv/bin/activate`

### install dependencies 
`pip install -r requirements.txt`

### copy .env.example and create .env and update below variable
`GEMINI_API_KEY`

### How to run 
`python <file_name>`

`python create_text.py`
